## Exercícios dos cursos:

## Pré-Requisitos

 * Ruby instalado.
 * GEMs: Bundler | Cucumber | RSPEC instaladas.
 * VSCode instalado.
 * Plugins do VSCode: Ruby | Gherkin Indent | VSCode Icons instalados.
 * Cucumber instalado.
 * Capybara instalado.
 * Selenium Webdriver instalado.
 * Drivers ChromeDriver, GeckoDriver e IEDriverServer.
